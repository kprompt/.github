# kprompt/.github

Organization profile and **default community health files** for [github.com/kprompt](https://github.com/kprompt).

## Profile

[`profile/README.md`](./profile/README.md) is rendered on the org home page.

## Defaults (apply to org repos without their own copy)

| File | Purpose |
|------|---------|
| [SECURITY.md](./SECURITY.md) | Vulnerability reporting |
| [CODE_OF_CONDUCT.md](./CODE_OF_CONDUCT.md) | Community standards |
| [CONTRIBUTING.md](./CONTRIBUTING.md) | How to contribute |
| [SUPPORT.md](./SUPPORT.md) | Where to get help |
| [`.github/ISSUE_TEMPLATE/`](./.github/ISSUE_TEMPLATE/) | Bug / feature forms |
| [`.github/PULL_REQUEST_TEMPLATE.md`](./.github/PULL_REQUEST_TEMPLATE.md) | PR checklist |

Repos that already define their own `.github/ISSUE_TEMPLATE` (or other health files) override these defaults.

See [GitHub docs: default community health files](https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file).
